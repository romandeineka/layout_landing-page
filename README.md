# Landing Page - Bike
This is an implementation of the landing page, according to [Figma design](https://www.figma.com/design/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0-1)

## This project utilizes the following technologies:
- HTML5
- CSS3 (SCSS)
- BEM-methodology
- Figma

## Landing page has the responsive layout for:
- Mobile (> 320px)
- Tablet 744px
- Desktop 1260px
